---
title: "MDX with gatsby-plugin-feed"
---

The [`gatsby-plugin-feed`](https://next.gatsbyjs.org/packages/gatsby-plugin-feed/) creates RSS feeds for your Gatsby site.

Out of the box, it works with the [`gatsby-transformer-remark`](https://next.gatsbyjs.org/packages/gatsby-transformer-remark) plugin. `gatsby-mdx` provides that same out-of-the-box functionality for MDX.

To use the MDX version of the default feed, do the following in your `gatsby-config.js` file.

```js
const mdxFeed = require("gatsby-mdx/feed");

module.exports = {
  plugins: [
    {
      resolve: `gatsby-plugin-feed`,
      options: mdxFeed
    }
  ]
};
```
